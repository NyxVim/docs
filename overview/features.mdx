## Introduction

NyxVim is built upon a foundation of carefully selected plugins and custom-built features. This guide will walk you through the key features that make NyxVim a powerful and user-friendly Neovim distribution.

<Info>
  NyxVim aims to provide a balance between functionality and performance, ensuring a smooth editing experience even with its rich feature set.
</Info>

## Core Features

### Base46: The Theming Engine

Base46 is NyxVim's high-performance theming plugin, offering a wide range of customization options.

<Accordion title="How Base46 Works">
1. Collects highlight groups from various sources
2. Processes overrides, new groups, and custom themes
3. Compiles everything into efficient bytecode
4. Lazy-loads integration files for optimal performance
</Accordion>

<Tip>
  You can dynamically reload UI-related options in your configuration file. This allows for real-time theme adjustments!
</Tip>

### Theme Switcher

NyxVim includes a theme switcher that integrates with telescope.nvim, allowing you to preview and switch themes on the fly.

### Statusline

NyxVim's statusline module offers four distinct styles to suit your preferences:

![NyxVim Statusline Styles](/images/statuslines.webp)

<Tip>
  You can customize the statusline further by modifying its components in your configuration file.

### Tabufline

The tabufline in NyxVim combines the functionality of a tabline and a bufferline:

- Manages both buffers and tabs
- Clickable buttons for easy navigation
- Each tab has its own set of buffers, similar to workspaces in modern operating systems

### Terminal Integration

NyxVim's terminal module provides powerful terminal management within Neovim:

- Create new terminal windows (floating, horizontal, or vertical)
- Toggle terminal windows on/off
- Dedicated terminal for running code
- Customizable window size, command, and highlight group for each terminal
- Telescope integration for managing hidden terminal buffers

<Accordion title="Detailed Terminal Features" icon="terminal">
- Create and manage multiple terminal windows
- Toggle terminal windows with customizable keybindings
- Dedicated terminal for running code snippets or entire files
- Flexible window options: floating, horizontal, or vertical splits
- Each terminal window can have its own:
  - Command to run
  - Window size
  - Highlight group for custom appearance
- Use the Telescope picker (`:Telescope terms`) to manage and unhide terminal buffers
  - Quick access with the keybinding `<leader>pt`
</Accordion>

<Tip>
  Access the terminal picker quickly with the keybinding `<leader>pt`. This allows you to easily manage and unhide terminal buffers.
</Tip>

<Info>
  The terminal module is designed to seamlessly integrate with your workflow, allowing you to run commands and manage multiple terminals without leaving Neovim.
</Info>

## Integrated Plugins

NyxVim incorporates several popular Neovim plugins, enhancing them with custom configurations and theming:

### Telescope.nvim

[Telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) is a highly extensible fuzzy finder:

![Telescope in NyxVim](/images/telescope.png)

### Nvim-tree.lua

[Nvim-tree.lua](https://github.com/kyazdani42/nvim-tree.lua) is a file explorer tree for Neovim:

![Nvim-tree in NyxVim](/images/nvimtree.png)

### Nvim-cmp

[Nvim-cmp](https://github.com/hrsh7th/nvim-cmp) is a completion plugin with multiple style options.

### LSP and Auto-completion

NyxVim uses [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) for Language Server Protocol support and [luasnip](https://github.com/L3MON4D3/LuaSnip) with [friendly-snippets](https://github.com/rafamadriz/friendly-snippets) for snippet completion.

### Additional Plugins

NyxVim also includes:

- [lazy.nvim](https://github.com/folke/lazy.nvim): A modern plugin manager
- [whichkey.nvim](https://github.com/folke/which-key.nvim): Displays possible key bindings
- [nvim-colorizer.lua](https://github.com/NvChad/nvim-colorizer.lua): Fast syntax highlighting
- [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter): Advanced syntax highlighting and code analysis
- [indent-blankline.nvim](https://github.com/lukas-reineke/indent-blankline.nvim): Indentation guides
- [gitsigns.nvim](https://github.com/lewis6991/gitsigns.nvim): Git integration for buffers
- [nvim-autopairs](https://github.com/windwp/nvim-autopairs): Automatic bracket pairing
- [mason.nvim](https://github.com/williamboman/mason.nvim): Package manager for LSP servers, DAP servers, linters, and formatters
- [conform.nvim](https://github.com/stevearc/conform.nvim): Lightweight formatter plugin

<Tip>
  Most of these plugins are lazy-loaded, ensuring that NyxVim starts up quickly despite its rich feature set.
</Tip>

## Conclusion

NyxVim combines powerful custom features with carefully selected and configured plugins to create a feature-rich, yet performant Neovim distribution. Whether you're a seasoned Vim user or new to the ecosystem, NyxVim provides the tools you need for an efficient and enjoyable coding experience.

<Info>
  Remember, most features in NyxVim are customizable. Don't hesitate to explore the configuration options to tailor NyxVim to your specific needs!
</Info>
