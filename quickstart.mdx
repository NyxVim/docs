## Prerequisites

- [Nerd Font](https://www.nerdfonts.com/font-downloads) as terminal font
  - Make sure the Nerd Font doesn't end with Mono to avoid small icons.
  - Example: JetbrainsMono Nerd Font and not ~~JetbrainsMono Nerd Font Mono~~
- [Ripgrep](https://github.com/BurntSushi/ripgrep) is required for grep search with Telescope (OPTIONAL)

### For MacOS
- Git
- Brew
- unzip

### For Linux (Ubuntu, Arch, Fedora, and WSL2)
- Git
- unzip

### For Windows
- Git
- unzip
- GCC, Windows users must have `mingw` installed and set in the path.
- Make, Windows users must have `GnuWin32` installed and set in the path.

### Install

<AccordionGroup>
  <Accordion icon="apple" title="MacOS">
    ```bash
    curl -L https://nyxvim.one/install | bash
    ```
    This command will install neovim, its peer dependencies and everything needed for your Better Vim setup.
  </Accordion>
  <Accordion icon="linux" title="Linux">
    ```bash
    curl -L https://nyxvim.one/install | bash
      ```
      This command will install neovim, its peer dependencies and everything needed for your Better Vim setup.
   </Accordion>
  <Accordion icon="windows" title="Windows">
    If you're using Command Prompt(CMD)
    ```bash
    git clone https://github.com/NyxVim/windows %USERPROFILE%\AppData\Local\nvim && nvim
    ```
    If you're using PowerShell(pwsh)
    ```bash
    git clone https://github.com/NyxVim/windows $ENV:USERPROFILE\AppData\Local\nvim && nvim
    ```
    **If the above path doesnt work, try any of these:**

      For CMD : %LOCALAPPDATA%\nvim
      ```bash
      C:\Users\%USERNAME%\AppData\Local\nvim 
      ```
      For PowerShell : $ENV:LocalAppData + "\nvim"
      ```bash
      C:\Users\$ENV:USERNAME\AppData\Local\nvim 
      ```
   Run `:MasonInstallAll` command after lazy.nvim finishes downloading plugins.
   Delete the `.git` folder from nvim folder.
   </Accordion>
</AccordionGroup>

### Uninstall

<AccordionGroup>
  <Accordion icon="apple" title="MacOS">
    ```bash
    curl -L https://nyxvim.one/uninstall | bash
    ```
    This command will uninstall neovim, its peer dependencies and everything needed for your Better Vim setup.
  </Accordion>
  <Accordion icon="linux" title="Linux">
    ```bash
    curl -L https://nyxvim.one/uninstall | bash
      ```
      This command will uninstall neovim, its peer dependencies and everything needed for your Better Vim setup.
   </Accordion>
  <Accordion icon="windows" title="Windows">
    ### Windows CMD
    ```bash
    rd -r ~\AppData\Local\nvim
    rd -r ~\AppData\Local\nvim-data
    ```
    ### Windows PowerShell
    ```bash
    rm -Force ~\AppData\Local\nvim
    rm -Force ~\AppData\Local\nvim-data
    ```
   </Accordion>
</AccordionGroup>
